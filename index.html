Tipos de datos

Entero

El tipo de dato entero se declara usando numb3r

Ejemplo:
numb3r variable;
number var = 12;

Flotante

El tipo de dato flotante se declara usando flo4t

Ejemplo:
flo4t flotante;
flo4t decimal = 3,1416;

Cadena

Para escribir cadenas se utiliza str1ng

Ejemplo:
str1ng cadena = "Soy una cadena";

Condicionales

Condición Si

Para declarar la condición si se utiliza 1f (condición)

Ejemplo:
1f (var > 2)
{ pr1nt("VAR es mayor que 2"); }

Condición Si no

Para declarar la condición si no, se utiliza 1f (condición) els3

Ejemplo:
1f (edad > 17)
{ pr1nt("Mayor de edad"); }
els3 { pr1nt("Menor de edad"); }

Condición switch

Declaración de interruptor, es un tipo de mecanismo de control de selección utilizado para permitir que el valor de una variable o expresión cambie el flujo de control de la ejecución del programa mediante búsqueda

Se utiliza sw1tch (identificador) {

c4se valor1:

Ejemplo: sw1tch (fruta) {

Ciclos condicionales

Ciclo For

El bucle for es una estructura de control en programación en la que se puede indicar de antemano el número máximo de iteraciones

Ejemplo:

Ciclo While

El propósito es repetir un bloque de código mientras una condición se mantenga verdadera.

Ejemplo: wh1le (i < 10) {
pr1nt("Estoy en un while");
}
